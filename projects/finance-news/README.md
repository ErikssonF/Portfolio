========================================
  Swedish Finance News Aggregator
========================================

HOW TO USE (SUPER SIMPLE!):
---------------------------

OPTION 1 - AUTOMATIC SETUP (RECOMMENDED):
1. Double-click "SETUP_AND_RUN.bat"
2. If Git Bash isn't installed, it will offer to install it for you
3. Click "Yes" and follow the simple prompts
4. The news will be fetched automatically!

OPTION 2 - MANUAL:
1. Double-click "RUN_FINANCE_NEWS.bat"
2. If you see an error, install Git for Windows manually (see below)
3. Run "RUN_FINANCE_NEWS.bat" again

That's it! The HTML file "finance-news.html" will be created in this folder.


FIRST TIME SETUP (if using manual option):
-------------------------------------------

If this is your first time and you chose the manual option:

1. Download from: https://git-scm.com/download/win
2. Install with default settings (just keep clicking "Next")
3. After installation, double-click "RUN_FINANCE_NEWS.bat" again


WHAT YOU'LL GET:
----------------

The script fetches the latest Swedish finance news from:
  • Dagens Industri
  • Avanza
  • Aftonbladet Ekonomi
  • Sveriges Riksbank

All news is compiled into a beautiful HTML report that opens in your browser.


TROUBLESHOOTING:
----------------

If the script doesn't work:
  → Make sure you have an internet connection
  → Install Git for Windows (link above)
  → Try running "RUN_FINANCE_NEWS.bat" again

If you see "finance-news.html" in this folder, it worked! Just open that file.
