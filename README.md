# Project Hub

A collaborative web platform for showcasing multiple people's projects in one unified site.

## Structure

- **Main Landing Page** (`index.html`) - Cards for each person/contributor
- **Individual Pages** (`pages/[name]/`) - Each person's project showcase
- **Shared Projects** (`projects/`) - All projects stored here

## Current Sections

- **Fredrik's Projects** - Web development projects including finance news aggregation

## How to Add a New Section

See [ADDING_SECTIONS.md](ADDING_SECTIONS.md) for detailed instructions.

## Projects

### Swedish Finance News
Real-time aggregated news from Swedish finance sources (DI, Avanza, Aftonbladet, Riksbank).
- Location: `projects/finance-news/`
- To update news: Run `RUN_FINANCE_NEWS.bat` in the project folder

Modern gradient theme with smooth animations, responsive card layouts, and consistent navigation across all pages.
